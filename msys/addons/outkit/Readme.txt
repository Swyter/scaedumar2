This is the source code for the Outposts! Kit.

In the .zip archive you will find the full source.
Outside are only the files with the code you need to add.

---- INSTALLATION ----
To install the kit:
- Look through the outkit_* files to see what is the code you need to add in your files.
- You can see an example implementation in the files in the .zip archive.
